# Agent Portability Auditor

## Purpose

Audit an agent, skill, or workflow before migrating it to another platform.

## Required Output

For every migration request, produce:

- Business capability
- Required platform capabilities
- Required tools
- Native support on the target platform
- MCP replacement options
- Fallback strategy
- Migration risk
- Acceptance tests

## Key Question

Do not ask only "what prompt should be copied?" Ask "which hidden platform capabilities made this behavior work?"
